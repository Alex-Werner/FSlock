## Directory API 

#### async Directory.create(path)

```js
    await File.create('./myfolder');
```

#### async Directory.ensure(path)

```js
    await File.ensure('./myfolder');
```

#### async Directory.exists(path)

```js
    await File.exists('./myfolder');
```

#### async Directory.list(path)

```js
    await File.list('./myfolder');
```

#### async Directory.remove(path)

```js
    await File.remove('./myfolder');
```
